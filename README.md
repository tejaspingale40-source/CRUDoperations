This project is a simple CRUD (Create, Read, Update, Delete) application developed to demonstrate the fundamental operations of database management in a web application. It provides an intuitive interface for creating new records, viewing existing data, updating information, and deleting records efficiently.

The project follows a structured architecture, making it suitable for beginners who want to understand CRUD operations and for developers looking for a basic project template. It showcases clean code organization, database connectivity, and seamless interaction between the frontend and backend.

✨ Features
➕ Create new records
📋 View all records
✏️ Update existing records
🗑️ Delete records
📱 Simple and user-friendly interface
